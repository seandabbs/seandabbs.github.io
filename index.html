NTSC Analog Television Signal Demonstration

This MATLAB script generates an animation showing an NTSC signal, in-phase and quadrature (IQ) components, and cathode ray tube (CRT) scan, all synchronized with each other. This was for a class project about demonstrating an image system of the student's choosing. My goal was to highlight how chrominance (color hue) is incorporated into analog television signals, which were originally designed only for luminance (brightness). By modulating the otherwise grayscale signal with color data (the "squiggles" on each step in the signal, except for the white [no color saturation] bar), color TV signals are backward compatible with black-and-white TVs, and black-and-white videos are forward compatible with color TVs. Running this script will simulate images of the user's choice; in this example I chose the SMPTE color test pattern. Some elements are exaggerated to better show details: each of the three beams isn't really a unique color or even visible, the phosphor grid would really have a smaller pitch, and signal voltage might be inverted depending on the stage at which it is measured.

Tileable Hexagons

A javascript/HTML/CSS tool that generates a tileable pattern of hexagons. I added the "torus" mode to demonstrate the wrap-around nature of the pattern. This wasn't made for anything specific.
